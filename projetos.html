Objetivos do Projeto

  • Criar um banco de dados para armazenar informações de usuários.
  • Desenvolver uma interface gráfica utilizando Apache NetBeans IDE.
  • Estabelecer uma relação entre o banco de dados e a interface gráfica.

Ferramentas Utilizadas

  • Apache NetBeans IDE: Ambiente de desenvolvimento para a interface gráfica.
  • Banco de Dados: XAMPP e phpMyAdmin para criação e conexão com o banco de dados.

Como Funciona o Projeto

O projeto consiste em um sistema de gerenciamento de dados com uma interface gráfica conectada a um banco de dados relacional. Cada grupo escolheu um dos temas propostos, relacionados aos Objetivos de Desenvolvimento Sustentável (ODS). O nosso tema, ODS3, envolveu a criação de um sistema educacional para Saúde Mental, incluindo banco de dados, triggers, documentação, fluxograma, logotipo e cronograma.
